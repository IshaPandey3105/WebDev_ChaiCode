UTF-8

UTF-8 (Unicode Transformation Format - 8-bit) is a character encoding standard used to represent text in computers and digital communication.
UTF-8 is a character encoding that uses 1-4 bytes to represent text, covering all Unicode characters. It's efficient, compatible with ASCII, and ensures proper display of diverse languages and symbols.

The Viewport

The viewport is the visible area of a webpage on a device's screen. It varies depending on the device (desktop, tablet, mobile).
